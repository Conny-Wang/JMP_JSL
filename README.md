# Git 使用帮助

1.克隆仓库到本地

git clone https://github.com/Conny-Wang/JMP_JSL.git

2.添加更新的文件

git add .

3.提交暂存区到本地仓库中:

git  commit -m “这里可以写备注”

参数设置修改文件后不需要执行 git add 命令，直接来提交

 git commit -a 

4.设置提交代码时的用户信息

git config --global user.name 'Conny'

git config --global user.email conny@conny.wang
